📝 Simple Tkinter To-Do List

A lightweight desktop To-Do List application built using Python and Tkinter.
This project demonstrates basic GUI development, user input handling, and dynamic list updates.

🚀 Features

Add new tasks

Delete existing tasks

View all tasks in a list

Simple and clean interface

Built using Python’s built-in Tkinter library

🛠 Tech Stack

Python 3

Tkinter (built-in GUI library)

No external dependencies required.

📦 How It Works

Tasks are stored in a Python set

The input field allows users to type a task

"Add Task" button inserts it into the list

"Delete Task" removes it safely using discard()

The listbox refreshes automatically after every action

▶️ How to Run
1️⃣ Make sure Python is installed

Check with:

python --version
2️⃣ Run the file
python main.py

(Replace main.py with your file name if different.)
